### 001 `.`
Normal mode
- `x` : delete one character
- `.` : repeat commands
- `u` : option one step backward
- `dd` : delete line
- `>G` : intend
- `j` : jump to next line

### 002 no repeating
- `a` : add content behind the cursor
- `A` : add content at the end of the line
- `j` : jump to next line
- `.` : repeat commands
